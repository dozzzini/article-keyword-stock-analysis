# 📈 뉴스 키워드 기반 미디어 심리가 SK하이닉스 주식 거래량에 미치는 영향 분석

**프로젝트 기간**: 2025.11.28 ~ 2025.12.02 (3일)  
**분석 대상**: SK하이닉스(000660)

---

## 🔍 프로젝트 배경

주식 시장에서 뉴스는 가격 변동뿐 아니라 **투자자의 실제 행동(매매)**을 직접적으로 유발하는 핵심 요인입니다.  
그러나 많은 텍스트 기반 금융 분석은 주가 방향 예측에 집중되어 있으며, 뉴스가 **얼마나 많은 투자자의 행동을 촉발했는지(거래량)**에 대한 정량적 검증은 상대적으로 없다고 인식하였습니다. 

---

## 🏭 기업 선정 이유: 왜 SK하이닉스인가

SK하이닉스는 다음과 같은 특성을 가진 기업으로, 미디어 심리와 거래량의 관계를 실험하기에 적합하다고 판단했습니다.

- 반도체 산업 특성상 **이슈 민감도와 거래량 변동성**이 큼
- 실적, 기술, 글로벌 정책 뉴스에 즉각적으로 반응
- 개별 기업 이슈와 산업 전반 이슈가 혼재되어 뉴스 영향력 분석에 적합

👉 따라서 본 프로젝트에서는 **미디어 정보가 실제 투자 행동으로 전환되는 과정을 관찰하기 위한 실험 대상**으로 SK하이닉스를 선정했습니다.

---

## 🎯 분석 목표

- 뉴스 기사 제목에서 핵심 키워드를 추출하고 키워드의 중요도와 감성 정보를 수치화하여 해당 점수가 **주식 거래량 변화와 어떤 관계를 가지는지** 탐색

---

## 🧠 흐름도

- **텍스트 처리**
  - KoNLPy를 활용한 형태소 분석
  - 빈도·중요도 기반 키워드 추출
- **감성 점수 설계**
  - KOSELF 기반 감성 사전 활용
  - 긍·부정 감성에 강도 가중치 부여  
    (부정 키워드에 더 큰 절댓값 → 손실 회피 심리 반영)
- **관련성 가중**
  - 기사 제목 내 ‘SK하이닉스’ 직접 언급 여부 반영
- **검증**
  - 뉴스별 `final_keyword_score`와  
    거래량(로그 변환) 간 관계 시각화

## ⚠️ 한계 및 향후 개선

본 프로젝트에서는 NLP 모델 학습 경험이 부족하여  
감성 분석을 **사전 기반 + 수동 가중치 방식**으로 설계했습니다.  
이로 인해 감성 점수의 **객관적 근거와 통계적 설명력에 한계**가 있음을 인식했습니다.

### 개선 방향
- **모델 기반 감성 분석 도입**  
  → 감성 분류 모델의 확률값을 점수로 활용하여 임의성 최소화
- **문맥 단위 분석 강화**  
  → 키워드 중심 분석에서 벗어나 Sentence-BERT 기반 기사/문장 임베딩 적용
- **데이터 기반 가중치 학습**  
  → 감성 키워드와 거래량 변화 간 회귀 계수를 가중치로 활용
- **시차(Time Lag) 반영**  
  → 뉴스 발생 이후 1·3·5일 거래량 변화 비교
